Gráfico 128 Comparativo Certificados Activos Acumulado — MoM por Período Barras agrupadas comparativas TD!P-S 692:701 chart23
Notación: CERTIFICADOS.Tabla3.(PERIODO[10 meses]; Estado Contrato; TIPO; Cuenta Id Solicitud) → TD.(P691:S701) — PERIODO / CERTIF_ACTIVOS / MES_ANTERIOR / DELTA / %_CAMBIO → Dashboard: "Comparativo certificados activos acumulado corte mismo día"
FUENTE ORIGEN
Sistema

Interfaz / Tabla
Reporte Certificados (Tabla3) → TD rango P691:S701 (tabla fórmulas)
Transformación
TD.(P691:Z701).
P=PERIODO · Q=CERTIF_ACTIVOS · R=VALOR_MES_ANTERIOR
S=VALOR_DOS_MESES_ANT · U=DELTA_MoM · X=PCT_CAMBIO
Definición
Comparativo mensual de certificados activos al corte del mismo día, mostrando el valor del mes actual junto al mes anterior y hace 2 meses. Columnas U y X calculan delta absoluto y % cambio MoM.
Llave de filtro
10 períodos (202507–202604) TIPO = REAL FLAG CIERRE MES = SI
Regla de transformación
Col Q = activos período N · Col R = activos período N-1 (offset +1) · Col S = período N-2.
Delta U = Q - R · % cambio X = (Q-R)/R.
Ejemplos: 202511→202512: +27 (+100%) · 202602→202603: +3 (+4.5%) · 202603→202604: -21 (-30%).
